# Zechery Acevedo - Giphy Project

## Overview: 
#   - Create a JavaScript based website utilizing an API key to pull Giphys from the Giphy main website.

## Details: 
#   - User may utilize the search bar to search for anything of their choosing to populate a result of 
#   --Giphy images to view.

## Technology Utilized:
#   - HTML / JavaScript / CSS

## User Stories
#   - As a user, I would like the ability to type into a search bar with a keyword to find a Giphy that I like.
#   - As a user, I would like those Giphys to be displayed in a list.
#   - As a user, I would also like to be able to go directly to the link of the Giphy.
